# HomezApp

# You will need Docker compose
## Install Docker compose if needed
### Windows
`winget install --id=Docker.DockerCompose -e`
__OR__
`winget install --id=RedHat.Podman -e`
## Start up containers
`docker compose up -d`
__OR__
podman compose up -d

## Navigate to locahost:4200 to get started
